## еҝ«йҖҹејҖе§Ӣ

### 1. е®үиЈ…дҫқиө–

```bash
# еүҚз«Ҝ
cd web && npm install

# еҗҺз«Ҝ (Go 1.21+)
cd backend && go mod download
```

### 2. й…ҚзҪ®

йҰ–ж¬ЎеҗҜеҠЁж—¶пјҢеҰӮжһңжІЎжңүжүҫеҲ°й…ҚзҪ®ж–Үд»¶пјҢдјҡиҮӘеҠЁеҲӣе»ә `~/.lumi/lumi.config.json`гҖӮ

дҪ д№ҹеҸҜд»ҘжүӢеҠЁеӨҚеҲ¶й…ҚзҪ®ж–Үд»¶:
```bash
mkdir -p ~/.lumi
cp backend/lumi.config.example.json ~/.lumi/lumi.config.json
```

### 3. ејҖеҸ‘жЁЎејҸ

```bash
# з»Ҳз«Ҝ 1: еҗҜеҠЁеҗҺз«Ҝ
cd backend && go run ./cmd/lumi

# з»Ҳз«Ҝ 2: еҗҜеҠЁеүҚз«Ҝ (зғӯйҮҚиҪҪ)
cd web && npm run dev
```

и®ҝй—® http://localhost:5173

### 4. з”ҹдә§жһ„е»ә

```bash
# жһ„е»әеөҢе…ҘејҸеҚ•ж–Үд»¶
cd web && npm run build:togo
cd ../backend && go build -o lumi ./cmd/lumi

# иҝҗиЎҢ
./lumi
```

и®ҝй—® http://localhost:3000

## й…ҚзҪ®иҜҙжҳҺ

### й…ҚзҪ®ж–Үд»¶дҪҚзҪ®

й…ҚзҪ®ж–Үд»¶жҢүд»ҘдёӢйЎәеәҸжҗңзҙўпјҲдҪҝз”Ёз¬¬дёҖдёӘжүҫеҲ°зҡ„пјү:

1. `./lumi.config.json` - еҪ“еүҚзӣ®еҪ•
2. `./lumi.json` - еҪ“еүҚзӣ®еҪ•
3. `~/.lumi/lumi.config.json` - з”ЁжҲ·зӣ®еҪ• (йҰ–ж¬ЎеҗҜеҠЁиҮӘеҠЁеҲӣе»ә)
4. `~/.config/lumi/config.json` - XDG й…ҚзҪ®зӣ®еҪ•

`publicServerURL` жҳҜеҸҜйҖүйЎ№гҖӮй…ҚзҪ®еҗҺпјҢиҝңзЁӢи®ҫеӨҮй…ҚеҜ№е‘Ҫд»Өдјҡдјҳе…ҲдҪҝз”ЁиҝҷдёӘең°еқҖпјӣжңӘй…ҚзҪ®ж—¶пјҢзі»з»ҹдјҡиҮӘеҠЁе°қиҜ•дҪҝз”ЁеҪ“еүҚжңҚеҠЎжңәеҷЁзҡ„еұҖеҹҹзҪ‘ IPпјҢиҖҢдёҚжҳҜй»ҳи®ӨеҶҷжҲҗ `localhost`гҖӮ

### Agent жқғйҷҗжЁЎејҸ

- `default`: ж•Ҹж„ҹж“ҚдҪңйңҖиҰҒз”ЁжҲ·зЎ®и®Ө
- `bypass`: иҮӘеҠЁжү№еҮҶжүҖжңүж“ҚдҪң (и°Ёж…ҺдҪҝз”Ё)

### и·Ҝз”ұи§„еҲҷ

- `@agent-id`: дҪҝз”Ё @ жҢҮе®ҡ Agent
- `keywords`: е…ій”®иҜҚеҢ№й…Қи·Ҝз”ұ
- `meta`: еҗҜз”Ёе…ғи·Ҝз”ұ (Agent еҸҜд»Ҙи·Ҝз”ұеҲ°е…¶д»– Agent)
